# React + Vite Project

## Setup do Ambiente

**Atenção:** Este projeto utiliza o **Node.js v20.18.0**. Por favor, verifique sua versão com `node -v` antes de prosseguir.

## Instalação e Execução

1.  **Instale as dependências:**
    ```bash
    npm install
    ```

2.  **Rode o projeto:**
    ```bash
    npm run dev
    ```

## IMPORTANTE:

ao rodar o projeto, aplicação deve estar rodando na porta :5173 (vite)
